## Prerequisites
- go 1.17 or earlier
- Recent browser
- Terminal

## Technologies
- Static: 
    - HTML5
    - CSS3
    - Javascript
- Dynamic:
    - Golang
## Installation 
1. Download the whole project `Travel` on your system
2. Open your terminal in `Travel`
```
cd Travel
```


```
http://localhost:8080
```
5. Here you are! Welcome in the main page of the Web application
